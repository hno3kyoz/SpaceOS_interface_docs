# 系统状态

本部分接口用于获取SpaceOS驾驶系统的状态。

`/space/info`

| 内容     | Key            | Value | 格式  | 单位 | 范围 | 示例 |
| -------- | -------------- | ----- | ----- | ---- | ---- | ---- |
| 系统时钟 | vehicle_length |       | float | 米   |      |      |
| 系统状态 | vehicle_width  |       | float | 米   |      |      |
| 错误编码 | vehicle_height |       | float | 米   |      |      |
